Role

Lead Front-End Developer. Responsible for HTML/CSS/JavaScript (AngularJS).

Description

Responsive web/mobile application built with AngularJS. This is a productivity tool used by 3rd party vendors in the field. The app exposes scheduling and document information to help the user perform their job in the field.

A key feature of this application is a signature capture capability that supports both mouse and touch input. Also featuring is a client-side (JavaScript) PDF generator that dynamically converts a web form into a PDF document (including signatures).
